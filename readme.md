# Installation

1. Make sure you have the following installed:
   -Node.js v22.14.0
   -nvm

2. Git clone this repository to your local machine
3. Use nvm to ensure Node.js version is correct:\
   `nvm install 22.14.0`\
   `nvm use 22.14.0`\
   Confirm the installation with:\
   `node -v`\

4. Install all dependencies:
   `npm install`
